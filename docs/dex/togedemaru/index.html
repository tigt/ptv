Togedemaru

Togedemaru sheds its spikes each new season, leaving metal thorns behind. Stepping on these spikes hurts worse than a Lego, so Trainers should use a magnet brush to clean Togedemaru regularly. In the wild, the spikes are gobbled up by Geodude.
