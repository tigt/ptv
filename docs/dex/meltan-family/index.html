Tools and silverware should be locked away during Meltan infestations, as they can squeeze through the smallest holes. The hex nut can’t change shape, so it’s often left behind. If you find the nut, put it in a wooden bowl and Meltan will get stuck retrieving it. Make sure not to put too many Meltan in the same bowl.
